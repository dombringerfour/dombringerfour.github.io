Git常用命令

一、

自报家门:git config --global user.name "name"//自报名字

自报家门:git config --global user.email "mail"//自报邮箱

二、

初始化一个git仓库:git init

三、

准备文件:git add 文件名//传单个文件

准备文件:git add .//传目录下所有文件

四、

存入仓库:git commit -m "说明"//将准备好的文件存入仓库并添加说明

五、

查看仓库操作信息:git reflog//可查看操作：reset为恢复，commit为存入数据备份。可查看commit的版本号(最前面的一串字符串)

六、

恢复数据:git reset --hard 版本号//可回退到对应版本号时的数据
